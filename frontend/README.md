# AI-OS Frontend

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Set your backend API URL in `.env`:
   ```
   REACT_APP_API_BASE_URL=https://your-backend.onrender.com/api/v1
   ```

3. Run locally:
   ```
   npm start
   ```

4. Build for production:
   ```
   npm run build
   ```
